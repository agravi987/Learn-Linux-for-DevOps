## 🛠️ Topics Covered So Far

- ✅ How the Internet works (DNS, IP, HTTP, etc.)
- ✅ Linux fundamentals & OS concepts
- ✅ File & directory navigation (`pwd`, `ls`, `cd`)
- ✅ File management (`cp`, `mv`, `rm`, `touch`, `mkdir`)
- ✅ File viewing (`cat`, `head`, `tail`, `less`)
- ✅ Text processing (`cut`, `sort`, `wc`, `tee`, `diff`)
- ✅ Links in Linux (hard links & soft links)
- ✅ `vi` editor essentials
- ✅ Remote access with `ssh`
- ✅ Disk & memory monitoring (`df`, `du`, `free`)
- ✅ Process management (`ps`, `top`, `kill`, `nohup`)
